Programming

“Computer science education cannot make anybody an expert programmer any more than studying brushes and pigment can make somebody an expert painter.”
- Eric S. Raymond

Intro to Computers and Programming

Much of using a computer may be considered intuitive, or relatively easy for the average user to figure out. This course teaches skills to take computer usage one step above average, covering things like filesystem management and navigation through command line interface, troubleshooting and problem solving, basic programming skills, and itroduces us to Github!

    Some key takeaways I took from this course include:
  • CLI and .bat files are powerful tools for assisting and automating file management.
  • Flowcharts are powerful problem-solving tools.
  • Many programming languages typically use similar constructs, just different syntax.


Programming Fundamentals

This course takes what we learned in Intro to Computers and Programming and builds upon the basic programming skills learned, strengthening our fundamentals through pseudocode, HTML, and Javascript

    Some key takeaways I took form this course include:
  • Code should be thoughtfully written, so that it's not only functional, but also accessible and readable.
  • Pseudocode is a great tool when developing software as it can be appplied to any language.
  • Loops and IF/THEN statements are critical to understand for writing effective code.

Python Programming

This course acts as our first real introduction to Object Oriented Programming, focusing on basic python skills, importing and using libraries, and creating and using objects and classes.

    Some key takeaways I took from this course include:
  • Object oriented programming can improve efficiency in both writing and execution of written programs.
  • Python is an extremely versatile programming language with countless "add-on" libraries available to use.
  • Classes are used like a "blueprint" for creating objects, further streamlining an object-oriented programming methodology.

Machine Learning (Course still in progess, will likely update when done.)

This course delves into foundational machine learning techniques designed to analyse data, going over machine learning model fundamentals, training and testing techniques, and which algorithms to use to suit your model to your needs.

    Some key takeaways I took from this course (so far) include:
  • There are three types of model training, supervised, unsupervised, and reinforcement.
  • How you train your model and what algorithm it uses will differ depending on use case.
  • Machine Learning models can be a powerful tool for predictive analysis, but should always be verified or accepted with a tolerance of failure.
